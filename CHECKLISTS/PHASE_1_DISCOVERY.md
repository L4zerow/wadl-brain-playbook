# Phase 1: Discovery Checklist

## Pre-Work
- [ ] Initial contact made
- [ ] Discovery call scheduled
- [ ] Basic company research completed
- [ ] Industry context gathered

## Discovery Call
- [ ] Company background understood
- [ ] Key stakeholders identified
- [ ] Current state documented
- [ ] Pain points discussed
- [ ] Success criteria explored
- [ ] Data sources mentioned
- [ ] Call recorded/notes taken

## Data Source Mapping
- [ ] All data sources identified
- [ ] Access status confirmed for each source
- [ ] Data quality assessed (clean/messy/unknown)
- [ ] Data freshness understood
- [ ] API documentation obtained (if applicable)

## Pain Point Analysis
- [ ] Pain points listed with severity (1-5)
- [ ] Frequency documented (daily/weekly/monthly)
- [ ] Affected users identified
- [ ] Current workarounds documented
- [ ] Impact quantified (time/money/frustration)

## User Profiling
- [ ] User types identified
- [ ] Technical levels assessed
- [ ] Primary needs documented
- [ ] Current solutions understood
- [ ] Access patterns considered

## Success Criteria
- [ ] Client's definition captured
- [ ] Measurable outcomes defined
- [ ] Current state baseline established
- [ ] Target state specified
- [ ] Measurement method agreed

## AI Opportunity Mapping
- [ ] Tier 1 (Quick Wins) identified
- [ ] Tier 2 (High Value) identified
- [ ] Tier 3 (Strategic) identified
- [ ] Impact/Effort assessed for each
- [ ] Confidence levels assigned
- [ ] Recommended tier selected

## Deliverable
- [ ] DISCOVERY.md created using template
- [ ] All sections completed
- [ ] Supporting research attached
- [ ] Recommendation clearly stated
- [ ] Investment estimate provided
- [ ] Risks identified

## Handoff
- [ ] Document reviewed internally
- [ ] Presented to client
- [ ] Questions answered
- [ ] Decision on next phase obtained
- [ ] If yes: Phase 2 kickoff scheduled

## Phase Exit Criteria
**Phase 1 is complete when ALL of the following are true:**
- [ ] Discovery Doc is finished and validated
- [ ] Client has reviewed and approved findings
- [ ] Recommendation has been made
- [ ] Go/No-Go decision received
- [ ] If Go: Phase 2 scope agreed
